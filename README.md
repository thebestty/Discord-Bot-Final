Name: WelcomeMaster

Type: Welcome Bot

Event Feature: Welcomes new users.

Who will use it? New people to the server can use the bot and ask it questions about the different channels.

What does the bot do? The bot will welcome a new member of a server and tell the user all the channels and tell them what they do. Then it will announce it on the main/general server that “User” has joined, and you should greet them.

Why would someone use this bot? People will use this bot because it helps them understand what each channel does, and announces that the user has joined, which tells them to greet the user, potentially becoming friends with the new user.

Four commands the bot will have: /help /showaround /answerquestions /announce

Safety feature: My bot will prevent the user from saying bad words, and they may be punished if they continue to do so.
